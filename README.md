# PzEnglishCard - Telegram-бот для изучения английского языка

## Описание
Telegram-бот, помогающий пользователям учить английские слова через интерактивные викторины. Поддерживает добавление персональных слов и их удаление.

## Функционал
- Викторина: случайный вопрос с 4 вариантами ответа
- Добавление собственных слов
- Удаление личных слов
- Хранение данных в PostgreSQL

## Технологии
- Python 3
- pyTelegramBotAPI
- PostgreSQL + psycopg2
- python-dotenv (для безопасного хранения токенов)
